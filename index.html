Stories & maps
Maps make important information accessible and really easy to understand.

Data explained
It's not enough to make important information available to download as a .csv or spreadsheet. Data must be put into context and explained. Our platform combines multimedia, narrative and beautiful maps to turn data into exciting, sharable content.

Manage map features

We believe that data points can be better. That's why we've build a tool for assembling map features with rich content. Add video, image galleries, tweets, source links, paragraphs of text, etc. Any content found on the web can be added to Orbitist maps. Oh yeah, did we mention you can manage your content from anywhere on any device?

Data visualized
Orbitist integrates with amazing data visualization tools like CartoDB and Mapbox's Tilemill. You have your favorite mapping tools. Odds are they can be used with Orbitist!
